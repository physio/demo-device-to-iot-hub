# MQTT to CSV

## Description

This is a demo project for show as you can read a json object from a mqtt broker (where you are subscribed), and save this into a `csv` file.

## Installation

```bash
$ npm install
```

Then rename `env-example` file in `.env`.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
